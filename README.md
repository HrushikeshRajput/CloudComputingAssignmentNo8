# Student Database using SQLite

## Objective
To create a simple database of one student using SQLite.

## Files
- schema.sql → Creates the student table
- data.sql → Inserts one record
- queries.sql → Performs SELECT and UPDATE queries

## Steps to Run
1. Open DB Browser for SQLite or sqliteonline.com  
2. Execute schema.sql  
3. Execute data.sql  
4. Execute queries.sql
